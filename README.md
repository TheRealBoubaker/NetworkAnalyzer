#Network Analyzer

## Module 1 : LibPcap Robot (Anis)
    + Read Interfaces
    + filter packets (HTTP)
    

## Module 2 : Packet Viewer (Iheb)
    + List packets
    + Filter pacacket by : IP, MAC, FROM/TO
    + Show Stats : Load, Bandwidth, Latency, BER over a period

## Module 3 : Session Viewer (Meriem)
    + List packets of the same session (given start date and start/end IP)
    + show Sequence diagram
    + Show Stats : Load, Bandwidth, Latency, BER over a period for the session
    + Show User Requests/Responses
    

## Module 4 : Dashboard (Amir)
    + authenticiation
    + select interfaces
    + selection session
    + show realtime graphs
    + show history graph
